# 中国风塔防游戏 — 开发计划

## 概述
类似部落冲突的中国风塔防游戏（网页版，p5.js）

## 阶段一：可玩原型（今天）
- [ ] 网格地图 + 中国风背景（山水、云纹）
- [ ] 3 种防御塔：箭塔、炮台、冰塔
- [ ] 点击放置/升级/卖出塔
- [ ] 敌兵波次（从地图边缘进入）
- [ ] 金币系统（杀敌掉落）
- [ ] 生命值系统
- [ ] 基础 UI：金库、血量、波次显示

## 阶段二：进阶玩法
- [ ] 城墙系统（路径规划）
- [ ] 更多塔类型（连弩、火塔、雷电塔）
- [ ] 更多敌兵类型（盾兵、骑兵、BOSS）
- [ ] 关卡系统
- [ ] 音效
- [ ] 存读档

## 阶段三：完整游戏
- [ ] 大本营升级系统
