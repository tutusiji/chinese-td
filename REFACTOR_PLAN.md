# 墨守重构计划

## 目标

把当前原型重构为以攻城、守城、生存建造为核心的中国风塔防游戏。整体参考《部落冲突》的清晰视角、建筑升级反馈、兵种职业识别和资源建造节奏，但视觉语言保持中国关隘、山水、城墙、机关武器的方向。

## 第一阶段：布局与地图面板

- 固定应用壳尺寸，避免开始画面从小框跳到大框。
- 地图主体始终在剩余空间内居中显示，侧栏不影响画布中心感。
- 地图区、HUD、组件库使用同一套面板结构。
- 覆盖层只覆盖地图区，不重新撑开整体应用壳。

## 第二阶段：地图表现

- 地图 tile 类型扩展为山地、草地、树木、河流、道路、城墙、城池、可建造地块。
- 地形需要影响策略：河流不可建造，山地适合远程塔，草地适合建造，树林可作为障碍/资源。
- 在 `MapDefinition` 中加入 `terrain` 或 `decoration` 层，避免把地形写死在渲染器里。

## 第三阶段：攻城兵种

进攻方第一批职业：

- 步兵：便宜、基础、消耗火力。
- 骑兵：高速突破，怕冰塔和路障。
- 炮兵：远程拆墙，移动慢。
- 大力士：高血量肉盾，适合顶塔。
- 巫师：范围法术伤害，脆但爆发高。
- 攻城车：专门撞墙和城门。
- 投石车：远程攻城，威胁防御塔和城墙。

每个兵种需要一张专属 UI 图、一个战场小人造型、一组数值标签。

## 第四阶段：防守武器与升级造型

防守方第一批建筑：

- 箭塔：快速单体，对骑兵优秀。
- 炮台：范围溅射，对密集单位优秀。
- 冰塔：控制减速，辅助输出。
- 投石车：大范围慢速高伤。
- 火箭塔：远程爆发，适合打高价值目标。

每种防守武器至少做 3 个等级造型：

- 1 级：木石结构，低矮，轮廓简单。
- 2 级：砖石强化，加入金属件和更明确的武器结构。
- 3 级：城防重器，体积更高，旗帜/瓦顶/火光/冰晶等主题元素更强。

## 第五阶段：UI 组件库

新增组件库界面，用于统一展示：

- 攻城兵种 UI 图。
- 防守武器各等级 UI 图。
- 地形图块：山地、草地、树木、河流。
- 资源/生存建造入口：木材、石料、粮草、金币。
- 面板、按钮、进度条、升级态、禁用态。

组件库会作为开发期和美术验收界面，避免图标、卡片、颜色散落在各处。

## 第六阶段：生存建造循环

- 增加资源：金币、木材、石料、粮草。
- 城墙、城门、武器塔可升级，升级后造型变化。
- 地图上可放置资源建筑或采集点。
- 守城时通过击杀、采集、提前迎敌获得资源。
- 攻城时通过兵种组合和路线选择破坏敌方防线。
